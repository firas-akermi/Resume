Heart stroke prediction using Random Forest algorithm

I used the dataset that I found on Kaggle web site to train a random forest model that predicts heart stroke. At first, I performed an EDA to gain insight from the data. In a second step, I cleaned the data from missing values and outliers. Next I implemented a Scikit-Learn pipeline to standardize the data, to encode categorical variables and to fit the data to the model. Finally, to enhance the performance of the model I tuned some hyper-parameters by implementing a random search algorithm and the final model reached an AUC of 0.75. The tuned model was saved as a pickle file and was used in a later step to develop a streamlit web application that predicts heart stroke probability giving a set of features.
